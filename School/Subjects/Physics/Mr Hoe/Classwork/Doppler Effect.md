[[Physics]]
When there is a relative movement between an emitter and an observer, the observer will note an apparent change in wavelength
	Towards -> shortening wavelength
	Away -> stretching wavelength
Δλ / λ = v / c
Δλ = change in wavelength / m
λ = original wavelength / m
v = speed / ms$^{-1}$
