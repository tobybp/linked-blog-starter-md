#### Modelling Assumptions
- A particle
- Negligible air resistance
- Not powered
![[particlemodels|700]]
### Horizontal:
Acceleration aₓ = `0`
Initial velocity `uₓ = ucosθ`
Velocity at any point `vₓ = uₓ + aₓt = ucosθ`
Horizontal position `Sₓ = uₓt + 1/2aₓt$^2$  = Utcosθ`

### Vertical Motion:
Acceleration = `g`
Initial velocity =` usinθ`
Velocity at any point = `Vᵧ = uᵧ + aᵧ = Vsinθ - gt`
Vertical position = `Sᵧ = Uᵧt + 1/2aᵧt$^2$`
                     = `utsinθ - 1/2gt$^2$`
### Direction of flight
![[direction of flight|300]]
Vᵧ = utsinθ - gt
θ = tan$^{-1}$(Vᵧ / Vₓ)
### Time of Flight
![[timeofflight]]
- Let Sᵧ = 0 and solve for t
- Calculate time to max height, and double it
### Range
Maximum horizontal position
- Sₓ = time of flight x `ucosθ`
### Max Height
- Let Vᵧ = 0 and solve for t
	Sub t into Sᵧ
- Use Vᵧ$^2$ = uy$^2$ + 2aᵧSᵧ 
	let Vᵧ = 0
